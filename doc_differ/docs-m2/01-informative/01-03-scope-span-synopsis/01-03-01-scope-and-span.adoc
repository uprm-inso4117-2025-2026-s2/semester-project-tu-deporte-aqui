==== 1.3.1 Scope and Span

===== Purpose

This section defines the boundaries, coverage limits, and operational constraints of Tu Deporte Aquí. Clearly defining scope prevents scope creep, ensures alignment across teams, and establishes realistic expectations during implementation.

The project has transitioned from documentation and system definition into implementation. The scope now focuses on building a working system while maintaining the constraints previously defined.

===== In-Scope Coverage

The system scope includes:

===== Local Puerto Rico Sports Leagues

- Coverage is limited to local leagues within Puerto Rico.
- National and international leagues are excluded.
- Emphasis remains on leagues where official APIs may not exist.

This constraint reflects the availability and reliability of data sources.

===== Data Handling Under Uncertainty

The system must handle:

- Delayed updates
- Conflicting reports
- Missing information
- Corrected or revised results

These cases are now considered during implementation, not only as documented scenarios.

===== Mobile-First Platform

The system is implemented as a mobile-first web platform.

This includes:

- Compact information display
- Clear status indicators
- Prioritization of essential game data
- Responsive layout across devices

===== Core Implementation Scope

The current milestone includes:

- UI implementation using Next.js
- Backend API routes
- Integration with Supabase
- Authentication and protected routes
- Display of real data in the frontend

This defines the transition from system definition to a working application.

===== Out-of-Scope Elements

The following elements remain out of scope:

- Real-time data updates (e.g., polling or WebSockets)
- Automated data scraping mechanisms
- National or international league coverage
- Advanced performance optimization
- Complex user roles or admin systems

These exclusions limit system complexity and keep the focus on core functionality.

===== Operational Constraints

Tu Deporte Aquí operates under the following constraints:

===== Informal Data Sources

- Data may originate from websites, reporters, or social media.
- Sources may conflict or provide delayed updates.
- Official verification mechanisms may not exist.

===== Limited Structure and Standardization

- Local leagues may lack standardized reporting formats.
- Inconsistent update timing must be expected.
- Historical corrections may occur without formal notice.

===== Transparency Requirement

Because of data uncertainty, the system must:

- Communicate provisional states
- Display last update times
- Indicate possible staleness
- Support traceability of changes

These constraints affect how data is presented and interpreted in the system.

==== Boundary Summary

Tu Deporte Aquí is a system focused on:

- Local Puerto Rico sports
- Handling unreliable or inconsistent data
- Providing transparent information to users
- Maintaining clear scope boundaries during implementation

These boundaries guide development decisions and help ensure consistent progress across teams.
