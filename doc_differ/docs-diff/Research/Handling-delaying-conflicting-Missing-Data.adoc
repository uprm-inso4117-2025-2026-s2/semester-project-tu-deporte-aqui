== [.inserted]#2. Handling Conflicting Scores Across Sources#

[.inserted]#Neither the NBA nor ESPN publicly display conflicts between sources.#

[.inserted]#Observed Industry Strategy:#

* [.inserted]#Use a primary trusted data provider.#
* [.inserted]#Do not display multiple score versions.#
* [.inserted]#Update scores only after verification.#
* [.inserted]#Perform silent backend reconciliation (users do not see conflicts).#

[.inserted]#Key Insight:#

[.inserted]#Major platforms resolve discrepancies internally before updating the user interface. Conflicting information is never exposed to users.#
