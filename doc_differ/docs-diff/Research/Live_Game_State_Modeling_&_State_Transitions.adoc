== [.inserted]#4. Standardized Game State Definitions#

* [.inserted]#*Scheduled*  #
[.inserted]#  Game confirmed but not started.#

* [.inserted]#*Live*  #
[.inserted]#  Game actively in progress with real-time updates.#

* [.inserted]#*Halftime*  #
[.inserted]#  Official break between regulation periods.#

* [.inserted]#*Overtime*  #
[.inserted]#  Additional period if regulation ends in a tie.#

* [.inserted]#*Delayed*  #
[.inserted]#  Temporary interruption (e.g., power outage, weather).#

* [.inserted]#*Suspended*  #
[.inserted]#  Game paused indefinitely with possible later resumption.#

* [.inserted]#*Final*  #
[.inserted]#  Game officially completed with confirmed result.#

* [.inserted]#*Canceled*  #
[.inserted]#  Game will not be played.#

* [.inserted]#*Postponed*  #
[.inserted]#  Game rescheduled before starting.#
