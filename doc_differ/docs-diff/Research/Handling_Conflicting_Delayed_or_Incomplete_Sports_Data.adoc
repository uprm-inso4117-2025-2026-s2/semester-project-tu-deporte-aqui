== [.inserted]#🧩 Common Strategies That Work in Practice#

=== [.inserted]#1. Source Trust Levels (Source Prioritization)#

[.inserted]#*What it is:*  #
[.inserted]#Treat data sources as *ranked*, not equal.#

[.inserted]#Example tiers:#

* [.inserted]#Official league source  #
* [.inserted]#Verified partner  #
* [.inserted]#Trusted reporter  #
* [.inserted]#Community / social media  #

[.inserted]#*How it helps:*#

* [.inserted]#Resolves conflicts deterministically#
* [.inserted]#Prevents low-quality data from overriding trusted sources#

[.inserted]#*Tu Deporte Aquí application:*#

* [.inserted]#Maintain a *Source Registry* with trust tier and expected latency#
* [.inserted]#Prefer higher-tier data when conflicts occur#
* [.inserted]#Retain lower-tier values for auditing#

[.inserted]#*Pros (PR context):*#

* [.inserted]#Works well with mixed official and informal coverage#

[.inserted]#*Cons:*#

* [.inserted]#Requires ongoing maintenance#

=== [.inserted]#2. Timestamps and Freshness Rules#

[.inserted]#*What it is:*  #
[.inserted]#Track:#

* [.inserted]#Event time (when it happened)#
* [.inserted]#Ingest time (when received)#

[.inserted]#*How it helps:*#

* [.inserted]#Makes delays visible#
* [.inserted]#Avoids showing stale data silently#

[.inserted]#*Tu Deporte Aquí application:*#

* [.inserted]#Display “Last updated at HH:MM”#
* [.inserted]#Mark data as *stale* after threshold#
* [.inserted]#Keep last known value#

[.inserted]#*Pros:*#

* [.inserted]#Supports transparency#

[.inserted]#*Cons:*#

* [.inserted]#Requires careful handling of missing timestamps#

=== [.inserted]#3. Quorum and Reconciliation#

[.inserted]#*What it is:*  #
[.inserted]#Allow disagreement between sources but:#

* [.inserted]#Require agreement to mark data as confirmed#
* [.inserted]#Keep multiple versions temporarily#

[.inserted]#*Why it works:*  #
[.inserted]#Preserves availability during failures.#

[.inserted]#*Tu Deporte Aquí application:*#

* [.inserted]#Multiple trusted sources agree → *confirmed*#
* [.inserted]#Disagreement → *unconfirmed / provisional*#
* [.inserted]#Reconcile later#

[.inserted]#*Pros:*#

* [.inserted]#Handles conflicts gracefully#

[.inserted]#*Cons:*#

* [.inserted]#May delay confirmation#
