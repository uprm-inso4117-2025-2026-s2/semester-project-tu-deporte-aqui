== [.inserted]#1. Introduction#

[.inserted]#Tu Deporte Aquí combines sports information from multiple leagues, reporters, and informal sources in Puerto Rico.#

[.inserted]#Because athlete names may appear:#

* [.inserted]#Duplicated#
* [.inserted]#Misspelled#
* [.inserted]#Formatted differently#
* [.inserted]#Incomplete#

[.inserted]#the system faces a significant *identity resolution problem*.#

[.inserted]#Without a structured strategy, the platform risks:#

* [.inserted]#Merging two different athletes into one profile#
* [.inserted]#Creating duplicate profiles#
* [.inserted]#Misattributing statistics#

[.inserted]#The purpose of this document is to evaluate identity resolution strategies and recommend a reliable approach.#

---

== [.inserted]#2. Current Problems#

[.inserted]#Common issues when merging athlete data include:#

* [.inserted]#Accent differences#
* [.inserted]#Nicknames#
* [.inserted]#Abbreviations#

---

== [.inserted]#3. Methods Evaluated#

=== [.inserted]#I. Exact Matching / Strict Matching#

[.inserted]#Records are linked only if all attributes match exactly.#

==== [.inserted]#Pros#

* [.inserted]#Low risk of false matches#
* [.inserted]#Easy to implement#
* [.inserted]#Low computational cost#

==== [.inserted]#Cons#

* [.inserted]#High rate of duplicate profiles#
* [.inserted]#Cannot handle misspellings or nicknames#

---

=== [.inserted]#II. Fuzzy Matching#

[.inserted]#Records are linked if they match within a certain tolerance (similarity threshold).#

==== [.inserted]#Pros#

* [.inserted]#Reduces duplicate profiles#
* [.inserted]#Handles small variations (misspellings, accents)#

==== [.inserted]#Cons#

* [.inserted]#Risk of false merges#

---

=== [.inserted]#III. Metadata Identifier Matching#

[.inserted]#Records are linked using additional identifying attributes (metadata).#

==== [.inserted]#Pros#

* [.inserted]#Lower risk of false merges#
* [.inserted]#More reliable than name-only approaches#

==== [.inserted]#Cons#

* [.inserted]#Depends on metadata quality#
* [.inserted]#Slightly more complex to implement#

---

== [.inserted]#5. Recommended Strategy#

[.inserted]#The most effective approach is a *layered strategy combining Metadata and Fuzzy Matching*.#

=== [.inserted]#Proposed Algorithm#

[.inserted]#1. Group athletes using metadata (e.g., team, league, position).#
[.inserted]#2. Within each group, apply fuzzy matching to detect duplicates.#
[.inserted]#3. Merge only when similarity exceeds a defined threshold.#

=== [.inserted]#Benefits#

* [.inserted]#Reduces duplicate profiles#
* [.inserted]#Minimizes false merges#
* [.inserted]#Handles real-world inconsistencies in naming#

---

== [.inserted]#Conclusion#

[.inserted]#A hybrid identity resolution strategy is essential for maintaining data integrity in Tu Deporte Aquí.#

[.inserted]#By combining metadata grouping with fuzzy matching, the system can:#

* [.inserted]#Improve accuracy#
* [.inserted]#Reduce duplication#
* [.inserted]#Prevent incorrect data associations#

[.inserted]#This approach balances reliability and practicality for a multi-source sports data platform.#
